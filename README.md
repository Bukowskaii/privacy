# Privacy policies

Published at <https://bukowskaii.github.io/privacy/>.

One directory per app, so each gets its own descriptive URL:

    /whale-shark-watch-face/   https://bukowskaii.github.io/privacy/whale-shark-watch-face/

Adding an app means adding a directory with an `index.html` and linking it
from the root `index.html`.

Each policy is a plain static page; `.nojekyll` keeps Pages from running a
build, so there is nothing that can fail between commit and publish.

Source of truth for the Whale Shark policy is `store/privacy-policy.md` in the
private `watch-faces` repository.
